📊 Property Management Dashboard (Power BI)
🚀 Overview

This project is an interactive Power BI dashboard designed to help property managers efficiently monitor and manage real estate operations across multiple properties. It provides a centralized and visual analytics platform for tracking key metrics such as maintenance, lease status, and property performance.

The dashboard enables data-driven decision-making, reduces manual reporting efforts, and improves operational efficiency.

🎯 Key Features
📍 Location Management – Track and compare performance across multiple properties
🛠️ Maintenance Monitoring – Identify pending and completed maintenance activities
📅 Lease Tracking – Monitor lease expirations and renewal timelines
🏢 Property Condition Analysis – Evaluate property status and operational health
📊 Interactive Visualizations – Dynamic charts and filters for real-time insights
⚡ Automated Reporting – Reduces manual effort and improves reporting speed
🧠 Impact

Reduced reporting time by ~30% through automation
Enhanced data accuracy by 80% after data cleaning and transformation
